# lip-init-server
Save pages in a grid with nodejs.

[![Build Status](https://travis-ci.org/valenz/lip-init-server.png?branch=master)](https://travis-ci.org/valenz/lip-init-server)

## Installation
	$ git clone git://github.com/valenz/lip-init-server.git

	$ cd lip-init-server

	$ npm install

	$ node app
